WebProjects:
Решение, содержащее различные шаблоны веб-проектов, автоматически
создаваемые Visual Studio 2015.
WebEmpty - шаблон Empty
WebEmptyMvc - шаблон Empty с добавленными ссылками для MVC
WebEmptyMvcWebApi - шаблон Empty с добавленными ссылками для MVC и Web API
WebApi - шаблон Web API
WebForms - шаблон Web Forms
WebMvc - шаблон MVC
WebSinglePage - шаблон Single Page

Lecture01/MvcSample:
Демонстрация создания базовых компонентов ASP.NET MVC - контроллеров
и представлений. Пояснение на примере структуры проекта MVC.

Lecture01/WebPictures:
Демонстрация дополнительных возможностей структуры проекта MVc на
примере сайта по "приему" картинок. Повторение некоторых возмож-
ностей HTML, CSS, JavaScript и AJAX.

Lecture02/Controllers:
Демонстрация различных способов написания контроллеров и действий,
различных видов результатов действий. Пример создания собственного
результата действия и собственного провайдера значений.

Lecture02/Filters:
Демонстрация всех видов фильтров MVC, примеры создания собственных
фильтров, примеры их работы в различных ситуациях. Демонстрация
порядка работы фильтров.
